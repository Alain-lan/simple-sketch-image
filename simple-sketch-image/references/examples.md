# Examples

Use these as compact patterns, not as fixed outputs. Adapt the source idea, composition, labels, and character rule to the user request.

## Body Illustration Example

User request:

```text
Use $simple-sketch-image to make a body illustration prompt for this paragraph: A good workflow does not depend on memory. It puts input, checks, and feedback where the agent can see them.
```

Good response shape:

```text
Image purpose: body illustration
Composition type: Workflow
Character rule: use default character 线框人 because the idea involves action
Core logic: visible inputs, checks, and feedback make the workflow reliable
Labels: 输入 / 检查 / 反馈 / 可见

Prompt:
Generate one standalone 16:9 horizontal simple sketch illustration for an article body illustration. Use a clean white background and minimalist black hand-drawn line art. Draw 线框人 placing three small cards labeled 输入, 检查, 反馈 onto a simple tabletop map labeled 可见. Use one sparse orange arrow moving across the cards and a small blue feedback loop returning to the table. Keep the image spacious and explanatory, with no title, no dense infographic, no 3D, no gradients, and no long text.
```
