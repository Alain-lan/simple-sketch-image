# Composition Types

Choose one primary type per image.

## Cover Main Visual

Use for article covers, blog headers, social previews, and opening images.

Draw one central metaphor that makes the topic recognizable quickly. Keep the image readable without dense labels. Leave enough empty area if external title text will be added later.

## Body Concept Illustration

Use for explaining one paragraph, claim, tension, or mental model inside an article.

Draw one scene or object that explains the idea. Prefer 3-6 short labels. Keep it calmer than a cover.

## Tutorial Steps

Use for how-to content and operational workflows.

Draw 3-5 step stations or panels. Keep each station simple. Use numbered labels only when needed. Do not squeeze a long manual into one image.

## Workflow

Use for input -> process -> output, AI work chains, content production, automation, and review loops.

Draw left-to-right or curved flow. Use sparse arrows. If a character appears, place the character at the key transformation point.

## Before and After

Use for messy -> organized, manual -> automated, vague -> clear, failed -> fixed, scattered -> structured.

Draw two contrasting zones with a clear transition. Avoid overloading either side.

## Problem and Solution

Use for pain points, bottlenecks, common mistakes, and remediation.

Draw the problem as a concrete obstacle, then show the fix as one decisive action or tool. Use red only for the problem highlight.

## Concept Metaphor

Use for abstract topics, strategy, cognition, systems, knowledge work, or hidden logic.

Turn the concept into a physical object or action: funnel, bridge, map, door, toolbox, filter, bookshelf, garden, machine, knot, ladder, scale, mailbox, checkpoint, or workshop.

## Structure Layers

Use for frameworks, hierarchy, stack, method layers, and dependency relationships.

Draw uneven hand-drawn layers or nested boxes. Avoid a polished pyramid or corporate slide look.

## Map or Route

Use for journeys, learning paths, product flows, writing pipelines, and decision paths.

Draw a simple path with a few landmarks. The route should show sequence and choice without becoming a complicated map.

## Mini Comic

Use for a short narrative, failure-to-success story, or relatable state change.

Use 2-4 panels. Each panel should contain one action. Keep labels very short.
